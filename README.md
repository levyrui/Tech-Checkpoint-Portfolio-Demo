# Tech Checkpoint Portfolio Demo

Este repositório foi criado no âmbito do módulo “Kit de Ferramentas para Trabalho Remoto e Freelance | Tech Checkpoint”.

Objetivo:
- organizar evidências de aprendizagem;
- publicar outputs produzidos durante as sessões;
- construir progressivamente um portfólio profissional digital.

Conteúdos:
- Sumários
- Exercícios
- Outputs produzidos com IA
- Evidências de aprendizagem
